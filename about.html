About


Hello! I'm Johnnie. I'm a third-year Computer Engineering student at Queen's University with a strong passion for mobile app development, web development, and high-performance computing (HPC). I’m particularly interested in building user-centric mobile apps using modern architectural patterns like MVVM and Clean Architecture. I enjoy optimizing app performance while ensuring a seamless and engaging experience for users.


In web development, I’m focused on creating scalable, secure full-stack applications using library like React.js, runtime environment like Node.js, and query language like GraphQL. I'm especially motivated to explore cloud services like AWS and Azure to enhance application performance and scalability.


With HPC, I’m passionate about optimizing algorithms for parallel and distributed computing to maximize efficiency in large-scale data processing and scientific simulations. I'm excited to dive deeper into frameworks like MPI, CUDA, and OpenMP as I continue learning.


Throughout my projects, I’m committed to integrating EDII-AS principles, ensuring my work is inclusive, equitable, and accessible to all users. Feel free to check out my work!
